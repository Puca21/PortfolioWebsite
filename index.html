Examining Different Procedural Generation Methods and Algorithms used in Game Development

For this project, I created a Procedurally Generated Maze in the Unreal Engine. The game was coded in C++ in Microsoft Visual Studio. The main objective of the thesis was to examine a number of Procedural Generation methods and Algorithms that are used in Game Development

The Player Character must traverse a set amount of randomly generated mazes while fighting off enemies.

The algorithm used to generate the random maze is called the ‘Recursive Backtrack Algorithm’. This algorithm works by choosing a starting point on the field. Randomly choose a wall at that point and make a passage to the adjacent cell, but only if the adjacent cell has not been visited yet. This then becomes the new current cell. If all adjacent cells have been visited, return to the last cell that has an unvisited cell next to it and repeat. The algorithm ends when the process has made its way back to the starting point.

Daily Sceal - Newspaper Website

For this college assignment, we were tasked to design a text based website that provided a news feed through a text-only environment. This site was not to include any other forms of media (pictures, video, flash, and java) and was to only be developed using HTML/CSS code and the Bootstrap front-end responsive framework. The site was to be seen as a front page and was to be considered as a black and white entity. This assignment was primarily a design exercise in typography and layout and was to embrace classic principles of design – balance, contrast, pattern, alignment, structure and so on.

Brainy - A Collection of Games.

The aim of this project is to examine game application development. It will particularly focus on HTML5, CSS and JavaScript as a way to create a cleanly designed app which flows between elements.

The main objective of the project is to create a brain training web application using JavaScript libraries, HTML and CSS. The app for this project, Brainy, focuses on individual games which aim to help the user improve their cognitive skills.

The end product will provide users with five games to choose from. The games contained in the app will be: memory game which is a matching game, a quiz with multiple choice answers and a countdown, Simon Says which consists of a sequence of ‘lights’ that the user must guess correctly, a numbers game where the user must find the computer-generated number and finally a hangman game where the user must guess a word that the computer randomly selects.
